# 📒 Contact Book

A Command-Line Contact Book application built using Python. This application helps users store, manage, search, update, and delete contact information efficiently. Contact data is stored permanently using JSON file handling.

## 🚀 Features

- Add New Contacts
- View All Contacts
- Search Contacts by Name or Phone Number
- Update Existing Contact Details
- Delete Contacts
- Persistent Data Storage using JSON
- User-Friendly Menu-Driven Interface

## 🛠️ Technologies Used

- Python 3
- JSON
- File Handling
- Functions
- Lists & Dictionaries


3. Run the program:
