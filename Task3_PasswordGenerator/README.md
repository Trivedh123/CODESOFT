# 🔐 Password Generator

A Python-based Command-Line Password Generator that creates strong and random passwords based on user preferences.

## Features

- Custom password length
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters
- Random password generation
- Input validation

## Technologies Used

- Python
- Random Module
- String Module

## How to Run

```bash
python main.py
```

## Example

```
========================================
      Password Generator
========================================
Enter password length: 8

Generated Password: 
!JN9pP%j
```
