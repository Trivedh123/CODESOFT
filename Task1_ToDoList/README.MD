# 📝 To-Do List Application

A simple Command-Line Interface (CLI) To-Do List application built using Python. This application helps users manage their daily tasks efficiently by allowing them to add, view, complete, and delete tasks. The tasks are stored permanently using file handling, ensuring data is retained even after the program is closed.

## 🚀 Features

- Add new tasks
- View all tasks
- Mark tasks as completed
- Delete tasks
- Persistent storage using a text file
- User-friendly menu-driven interface
- Input validation and error handling

## 🛠️ Technologies Used

- Python 3
- File Handling
- Lists and Dictionaries
- Functions
- Exception Handling
